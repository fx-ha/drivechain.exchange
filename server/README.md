# Server

## Getting started

Development:
- copy ```.env.example``` to ```.env```
-  ```npm install``` to install
-  ```npm run watch``` to compile
- ```npm run dev``` to start

Production:
- copy ```.env.example``` to ```.env```
-  ```npm install``` to install
-  ```npm run build``` to build
-  ```npm run start``` to start
