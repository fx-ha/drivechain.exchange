# Server

## Getting started

Development:
- copy ```.env.example``` to ```.env```
-  ```npm install```
-  ```npm run watch```
- ```npm run dev```

Production:
- copy ```.env.example``` to ```.env```
-  ```npm install```
-  ```npm run build```
-  ```npm run start```
