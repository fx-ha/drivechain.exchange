# Web
This is a [Next.js](https://nextjs.org/) project

## Getting Started
Development:
- copy ```.env.local.example``` to ```.env.local```
-  ```npm install```
- ```npm run dev```

Production:
- copy ```.env.local.example``` to ```.env.local```
-  ```npm install```
-  ```npm run build```
-  ```npm run start```
